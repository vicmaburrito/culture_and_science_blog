## Getting Started

To get a local copy up and running follow these simple example steps.

### Setup

Install gems with:

```
bundle install
```

Setup database with:

```
   rails db:create
   rails db:migrate
   rails db:seed
```

### Prerequisites

Ruby: 2.7.3
Rails: 6.1.4
Postgres: >=9.5

### Live Preview

### Usage

Start server with:

```
    rails server
```

Open `http://localhost:3000/` in your browser.

